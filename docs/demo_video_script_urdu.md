Aur bas chand secondon mein, ek AI-powered system aapki baat samajh leta hai, qareeb ke best providers dhund leta hai, fair pricing de deta hai, aur ek confirmed technician book kar deta hai — sab kuch automatically. ## SCENE 3 — LIVE DEMO: REQUEST BHEJEIN (1:00 – 1:45)

### 🎤 Script:
> *"Ab main aapko poora flow dikhata hoon. Main Roman Urdu mein ek natural language request type karta hoon..."*

### 📱 Screen Par — YEH LIVE KAREIN:
1. **Home Screen** par input field tap karein
2. Type karein: **`AC repair chahiye urgently, G-13 Islamabad`**
3. **Submit / Send** tap karein
4. **Agent trace panel** ka wait karein — har agent ko point out karein:
   - *"Dekhein — Intent Agent ne abhi 'AC Repair' service extract ki, 'G-13' location nikali, aur 'high' urgency samjhi."*
   - *"Complexity Agent ne is kaam ko intermediate classify kiya."*
   - *"Discovery Agent ne area mein providers dhund liye."*

> *"Yeh sab 2 second se kam mein hua — 3 AI agents mil kar kaam kar rahay hain."*

## 🎥 Recording Tips

| Tip | Details |
|---|---|
| **Screen Record Tool** | Laptop ke liye **OBS Studio** ya Android ka built-in screen recorder |
| **Audio** | Narration alag se quiet kamre mein record karein, editing mein overlay karein. Ya live narrate karein. |
| **Speed** | Jaldi mat karein — agent traces aur animations ko play hone dein, yeh impressive lagte hain |
| **Agents highlight karein** | Jab agents fire hon toh agent trace panel par zoom in ya circle karein — yeh aapka wow factor hai |
| **5 min se kam rakhein** | Upar ka script natural pace par ~4:30 ka hai |
| **Editing** | CapCut (free) ya DaVinci Resolve. Thori si background music daal dein polish ke liye. |
| **Resolution** | Kam se kam 1080p mein record karein |
